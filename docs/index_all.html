Configures a device polling function. ///

  • event_desc Event descriptor (return value of /toroco/device)
  • time Refresh time in seconds
  • converter Function for converting the raw value
wait_for_input (input_desc[, timeout])
Wait for an input. /// This function pauses a coroutine or trigger handler until an input is received.

Parameters:

  • input_desc Input descriptor (return value of /toroco/input)
  • timeout Number of seconds to wait or return nil. (optional)

Returns:

    List of extra parameters of the received event. If there is none, it returns nil.
send_output (output_values)
Send the behavior output. /// This function sends the output of a behavior, that is, it sends a signal for each output event of the behavior.

Parameters:

  • output_values Table with the extra parameters for each event.
set_output (output_values)
Set the behavior output. /// This function sets the output of a behavior, that is, it sets a signal for each output event of the behavior. If any of the output events is inhibited/suppressed and then released, it resends the signal to the targets.

Parameters:

  • output_values Table with the extra parameters for each signal.
unset_output ()
Set the behavior output. /// This function unsets the output of a behavior, that is, it unsets the signals for each output event of the behavior.
set_inputs (receiver_desc, inputs)
Set the behavior inputs. ///

Parameters:

  • receiver_desc Receiver descriptor (return value of /toroco/behavior)
  • inputs Table where the keys are the input names and the values are input descriptors (return value of /toroco/device or /toroco/behavior)

Sets the inhibitors for an emitter. ///

Parameters:

  • outputs_inhibitors Table of inhibitors for each output event.

inhibit (behavior, event_desc[, timeout])
Inhibit an event. /// This function inhibits an event sent by a behavior. The inhibition is independent of other inhibitions to the same event that were set by other behaviors.

Parameters:

  • behavior Inhibiting behavior.
  • event_desc Event descriptor (return value of /toroco/device or /toroco/behavior)
  • timeout Number of seconds. (optional)
release_inhibition (behavior, event_desc)
Release an inhibition to an event. /// This function releases an inhibition to an event. The released inhibition is independent of other inhibitions to the same event that were set by other behaviors.

Parameters:

  • behavior Releasing behavior.
  • event_desc event descriptor (return value of /toroco/device or /toroco/behavior)
suppress (behavior, event_desc, receiver_desc[, timeout])
Suppresses an event received by a specific behavior. /// The suppression is associated with the running behavior, and is independent of other suppressions to the same event that were set by other behaviors.

Parameters:

  • behavior Suppressing behavior.
  • event_desc Event descriptor
  • receiver_desc Receiver descriptor (return value of /toroco/behavior)
  • timeout Number of seconds. (optional)
release_suppression (behavior, event_desc, receiver_desc)
Release a suppression to an event /// This function releases a suppression to an event for a specific behavior. The released suppression is the one associated with the running behavior, and is independent of other inhibitions to the same event that were set by other behaviors.

Parameters:

  • behavior Releasing behavior.
  • event_desc Event descriptor (return value of /toroco/device or /toroco/behavior)
  • receiver_desc Receiver descriptor (return value of /toroco/behavior)
